# Student Database SQL Project

## 📌 Description
Ye project ek simple Student Database system hai jo SQL ka use karke banaya gaya hai. Isme student aur employee related tables aur queries hain.

## 🛠️ Technologies Used
- SQL
- MySQL

## 📂 Files Included
- Student.sql → Student table aur queries
- Employee.sql → Employee table aur queries
- Studnet1.sql → Additional queries
- student.db.sql → Database structure

## ▶️ How to Run
1. MySQL open karo
2. Database create karo:
   CREATE DATABASE studentdb;
3. Database use karo:
   USE studentdb;
4. SQL file run karo

## 🎯 Features
- Student data manage karna
- Employee data manage karna
- Basic SQL queries practice
